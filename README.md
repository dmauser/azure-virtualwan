## Contributing

When contributing new labs or scripts, please follow the project conventions:

- **Script conventions:** See [docs/SCRIPT_CONVENTIONS.md](./docs/SCRIPT_CONVENTIONS.md) for naming, structure, and error handling standards.
- **Lab README template:** Use [docs/LAB_README_TEMPLATE.md](./docs/LAB_README_TEMPLATE.md) as the starting point for new lab documentation.
- **Cleanup:** Every lab must include a cleanup script or instructions to remove provisioned resources.
